# LinkedIn-Automation

**Author:** Arnab Santra & Divyansh Garg

This project is a Python-based utility designed to fetch and store the HTML content of web pages. It is built with robustness in mind, handling modern security headers and providing clear error messaging.

---

## Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/arnabsantra365/LinkedIn-Automation.git\
cd LinkedIn-Automation
```

### 2. Prerequisites
Due to PEP 668 (Externally Managed Environments), it is recommended to run this project within a Python Virtual Environment to avoid system conflicts.

**Install the virtual environment tool (Linux/Debian):**
```bash
sudo apt update
sudo apt install python3-venv
```
# Create a virtual environment
```bash
python3 -m venv venv
```

# Activate the environment
# On Linux/macOS:
```bash
source venv/bin/activate
```
# On Windows:
```bash
.\venv\Scripts\activate
```
# Install dependencies
```bash
pip install -r requirements.txt
```
